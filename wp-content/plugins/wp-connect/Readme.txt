支持使用社交帐号登录网站，同步文章、评论到微博/SNS，把微博评论回推到网站，使用微信搜索WordPress文章等。

== Description ==

WordPress连接微博 插件于2011年1月20日发布第1版，距离至今两年有余，已经更新100多个版本。已被数万家网站使用，插件于2013年1月28日插件上线两年之后发布了“Wordpress连接微博 专业版”。

1. 使用QQ、新浪微博、微信等20个社交帐号登录您的网站。 
包括微信、QQ、新浪微博、腾讯微博、人人网、开心网、淘宝网、支付宝、豆瓣、天涯、百度、易信、360、Twitter、Google、Yahoo、facebook、LinkedIn、GitHub、MSN
2. 同步文章、评论到10个微博/SNS。 
包括新浪微博、腾讯微博、人人网，开心网、豆瓣、天涯微博、Twitter、facebook、LinkedIn、易信朋友圈等
3. 支持自定义APP key
4. 本地化“社会化评论框”，包括微博评论回推到网站和新回复邮件通知。
5. 社交数据统计
6. 同步全文或者部分内容到其他博客。
包括新浪博客、网易博客、人人网日志、开心网日记、点点网、豆瓣日记、Tumblr、LOFTER（网易轻博客）、QQ空间、百度空间 
（其中QQ空间、百度空间部分主机无法使用，如果强烈需要这两个的请谨慎购买。）
7. 使用社会化分享按钮。
8. 写文章/发微博WAP页面（适用于手机浏览器）
9. 隐藏文章的部分或者全部内容，用户通过登录、回复、分享等行为后才能显示隐藏的内容。
10. 支持wp后台自动升级。

= 常见问题 =

详细文档 https://wptao.com/doc/wp-connect.html

1、对服务器有什么要求?
需要安装zend组件，目前仅支持安装了PHP5.2.x、PHP5.3.x和PHP5.4.x系列的服务器环境，购买前，请打开 http://pan.baidu.com/s/1i5e4n57 下载后上传到网站根目录，通过 http://您的网站网址/test.php 打开，检查您的服务器是否支持！

2、是一次性收费还是每年收费？我能免费升级吗?

是一次性收费，下载的插件可以永久使用，免费升级一年，第二年起每年收取25%升级费用，不升级不收费。

3、我能退货吗?

该插件是互联网产品，具有可复制性，一旦购买，不能退款。

4、我用了一段时间后，能更换域名吗？

插件是绑定域名的，一旦购买后不能修改。

5、提供技术支持吗?

支持的，请联系插件作者即可。(联系方式见插件页面)

6、我应该选择单域名/多域名/WPMU中的哪一类型?
比如您的网址是 http://www.yoururl.com/blog/ 
如果您购买了单域名，只能在这个www.yoururl.com域名下的网站使用，不能用于二级域名，但是你可以直接为二级域名购买插件。提示：加www和不加www视为同一个单域名，都能使用。
如果您购买了根域名，支持该域名下的所有网站，所有的二级域名下的网站也能使用。
如果您购买了WPMU，在根域名的基础上，还能使用WordPress MU Domain Mapping插件绑定其他域名。

== Installation ==

1. Upload the `wp-connect` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the settings using "WP Connect" under Settings panel.

下载 WordPress连接微博 插件，上传wp-connect目录及其文件到 "/wp-content/plugins/" 插件目录，在后台管理中激活插件，到设置页面开启功能并设置，以及账号绑定等.

= 4.0 =

2015/5/8

这是一次全新的开始。当初（2010-12-27）开发插件时想法还不够成熟，很多用户数据直接保存在默认的数据库表wp_usermeta，
一旦用户多了，会导致该数据库表臃肿无比，对开发插件造成阻碍，更别说大数据分析了，纠结了一年多，终于下定决心完善插件，
花了一个星期整理，把社交数据全部无缝地转移到新的数据库表 wp_connect_user，可能会花您几分钟时间，但是绝对值得这么做！

新增:开始使用refresh_token延长授权期限，也就是同步微博帐号会自己更新授权，不过呢，万恶的新浪微博和linkedin不支持噢！
新增:在文章发布页面显示当前绑定的微博同步帐号。
新增:国外SNS使用代理（位于同步设置，包括登录和同步，覆盖Twitter、Facebook、Google等，绑定时记得翻墙~），如果您主机在海外（包括港澳台）不用开启。
新增:在您网站的任何地方登录或者绑定微博时，会更新当前微博帐号在整个网站的授权（包括WPMU）。
新增:记录社交用户的所在地、性别、认证信息等，将在下一版本显示并且统计出来。
新增:修改文章绑定的单条微博地址。（位于文章发布页面。描述：文章同步后，会在微博产生一条数据，从此它们确定了绑定关系，会影响到后续的同步评论和微博评论回推，当然，您也可以编辑它。）
新增:使用GitHub登录网站，也可以自己申请API
升级:使用Google/Yahoo登录升级为新的接口，也可以自己申请API
升级:社交数据全部无缝地转移到新的数据库表 wp_connect_user
优化:微博评论回推开始采用智能机制以提高准确性，以免浪费接口调用。（由于2015.3.31新浪微博接口升级后，接口调用总次数一般在每人每小时150次，插件需要用到的接口包括同步微博、微博评论回推，很多人会设置评论回推频率为每分钟，这就导致没几分钟接口次数就调用完了，直接导致不能同步微博，采用智能机制后，能减少接口次数浪费，也能提高准确率。）
优化:隐藏内容后，微博分享可见功能更加人性化了，如果文章同步过，将以【转发】的形式分享。

修正一些罕见的bug
